# ⚡ Começar Rapidamente - Questões Pré-Definidas

## 🚀 Solução Imediata

Agora tem 2 formas de começar:

### **Opção 1: Importar Questões Padrão (RECOMENDADO)** ⭐

1. Abra a aplicação
2. Vá a **"Questões"**
3. Clique no botão azul grande **"📥 IMPORTAR QUESTÕES PRÉ-DEFINIDAS"**
4. ✅ 12 questões são carregadas automaticamente para todos os 8 departamentos

**Depois pode:**
- Ir a **"Execução"** e começar a preencher auditorias
- Adicionar mais questões em **"+ Nova Questão"**

---

### **Opção 2: Adicionar Manualmente**

Se preferir customizar desde o início:

1. Clique **"+ Nova Questão"**
2. Preencha:
   - Departamento: (ex: DCO, DEN, DPR, etc)
   - Questão: (ex: "Existe registo de pedidos de compra?")
   - Foco: (ex: "Rastreabilidade")
3. Clique **"+ Adicionar"**
4. Repita para mais questões

## 🔄 Próximos Passos

### Após Importar:

1. **Ir a "Execução"**
   - Selecionar departamento
   - Preencher questionário
   - Registar constatações

2. **Editar Questões** (opcional)
   - Voltar a "Questões"
   - Adicionar mais questões se necessário
   - As existentes continuam guardadas

3. **Sincronizar com Google Sheet** (depois)
   - Quando o AppScript for redeployado
   - Os dados sincronizam automaticamente

## 💾 Dados Locais

- Todos os dados estão salvos em **localStorage** do browser
- **Não perdem** ao fechar ou recarregar a página
- Estão sincronizados com Google Sheets quando disponível
